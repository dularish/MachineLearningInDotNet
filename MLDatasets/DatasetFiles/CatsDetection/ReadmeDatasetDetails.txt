This dataset contains both testing and training data for cats detection

Training_X details :
Number of columns : 64 * 64 * 3; That is 64*64 image pixel data for 3 channels (red, blue, green)
Number of rows : 209 ; That is 209 samples

Training_Y details :
Number of columns : 209 ; That is 209 samples
Each value indicates if it's a cat image (represented by 1) and non-cat image (represented by 0)
